# Feedback Control - CMU MATLAB Assignments

This repository contains **control theory assignments from Carnegie Mellon University** implementing feedback control for complex dynamical systems. The work focuses on **pendulum-on-cart stabilization and tracking** using linear and nonlinear control techniques.

**Key Topics:** LQR (Linear Quadratic Regulator), state-space analysis, system controllability, feedback control design, and trajectory tracking. **Methods:** Linearized control, nonlinear dynamics with feedback, observer-based estimation, and feedforward control. **Simulations:** Includes MATLAB scripts for stability analysis, control gain optimization, and system response visualization. The PDFs contain detailed problem statements, derivations, and performance plots demonstrating successful controller implementation.
